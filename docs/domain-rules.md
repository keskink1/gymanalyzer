**Common**
1) Auditing will be handled in common library.

**User**

1) Email format must be valid.
2) Age must be older than 18
3) Password must be at least 6 characters, containing one capital letter, one number and one special character
4) First name and last name - each - must be at least 3 characters.
5) No null fields allowed.

**Exercises**
1) Only viable categories are allowed.
2) Both exercise name and description must be each 3 characters long.
3) Difficulty level must be between 1 and 10. 
4) No null fields allowed.

**Workouts**
1)  Workout name must be at least 3 characters.
2) Can only choose from existing exercises.
3) Set, repetition and weight count must be higher than 0. For each set, weight can be different.
4) Adding same exercise twice is allowed.
5) Can be assigned to any date, no restrictions.
